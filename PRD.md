# 데이터 분석 절차
## 1. 데이터 분석 목표
리조트 식음업장에서 수집된 과거의 메뉴별 판매 데이터를 기반으로, 향후 1주일간 각 메뉴의 예상 판매량을 예측하는 AI 모델을 개발

평가 산식: 식음업장 별 가중치가 있는 SMAPE
    s: 식음업장명
    w_s: 식음업장 s의 가중치(비공개)
    I_s: 식음업장 s에 속한 품목 컬럼 집합
    T_i: 품목 i에서 유효한 날짜 수(At,i != 0)
    A_t,i, P_t,i: 날짜 t, 품목 i의 실제값과 예측값

## 2. 데이터 탐색
### 2-1. 데이터 확인
훈련 데이터는 './train/train.csv', 테스트 데이터는 './test/' 폴더 내의 파일들, 제출 데이터 양식은 './sample_submission.csv'

### 2-3. 기술 통계량 및 데이터 포맷 확인

### 2-4. 결측치 및 데이터타입 이상 처리

### 2-5. 컬럼 별 그래프 시각화

### 2-6. 이상치 확인 및 대체
4분위 수 출력 및 이상치 데이터를 탐색하여 IQR 기반 제거/보정이나 통계량 대체(이산형 데이터이고 매출 건수이므로 지나치게 이상한 값만을 대상으로 함)
